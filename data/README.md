本目录包含了用到的数据集，读取数据的相关函数。

# 数据集命名规则
原始数据集名-负样本类别下标号 vs. 正样本类别下标号   
如：iris-0 表示类别下标为0的类作为负样本，其余全部作为正样本

# 数据集来源 
KEEL：https://sci2s.ugr.es/keel/category.php?cat=clas  
UCI：http://archive.ics.uci.edu/ml/datasets.php
